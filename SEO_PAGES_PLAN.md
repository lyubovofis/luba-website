## 4. TECHNICAL SEO REQUIREMENTS

### Schema Markup (for AI & Voice Search):
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [...]
}

{
  "@context": "https://schema.org",
  "@type": "Service",
  "name": "Работа с денежными блоками",
  "provider": {
    "@type": "Person",
    "name": "Любовь Лукащук"
  }
}

{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "Как убрать денежные блоки"
}
```

### Voice Search Optimization:
1. Include exact question in H1 or H2
2. Answer in first 50 words (featured snippet)
3. Use conversational language
4. Target position 0 (AI Overview)
